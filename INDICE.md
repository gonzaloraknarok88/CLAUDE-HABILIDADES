## USO RPIDO

### Para Claude.ai
1. Abre Claude.ai
2. Haz clic en el 铆cono de habilidades (З)
3. Busca una habilidad o carga una personalizada

### Para Claude Code
```bash
cp -r <skill-name> ~/.config/claude-code/skills/
claude
```

### Para API
```python
import anthropic
client = anthropic.Anthropic(api_key="your-key")
response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    skills=["skill-id"],
    messages=[{"role": "user", "content": "prompt"}]
)
```
